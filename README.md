# README

## 開発言語
- Ruby 3.0.1
- Rails 6.1.7

## 就職Termの技術
- gem/devise
- AWS EC2

## カリキュラム外の技術
- gem/rails_admin
- gem/cancancan
- gem/ransack

## 実行手順
````
$ git clone git@github.com:rikiya-miyazawa/Eduka_app.git
$ cd Eduka_app
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
````
